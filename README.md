# Quick Start
1. `git clone https://github.com/seemcat/magic-11ty.git`
2. `cd magic-11ty`
3. `npm install`
4. Grab your `Public Key` from your [**Magic Dashboard**](https://dashboard.magic.link) and paste it into header.njk.
`npm start`
